# Freactional Trainer

### RUNNING

1. Clone repo:
    
    ```sh
    git clone https://github.com/tikhonp/fractional-trainer.git
    cd fractional-trainer
    ```

2. Create python venv:
    
    ```sh
    python -m venv env
    . env/bin/activate
    ```

3. Install dependesies:
    
    ```sh
    pip install -r requirements.txt
    ```

4. Run:
    
    ```sh
    python main.py
    ```
